Esse foi um projeto simples para uma feira de projetos realizado pela minha escola.
A historia apresentada foi de que seu PC foi enfectado por um virus, ocultando todos os seus aplicativos menos o bloco de notas
e apartir dele você tem que conseguir acessar seu anti virus para que tudo volte ao normal.
Para aplicar é nescessario o uso da internet. 

O intuito é apresentar conceitos basicos de programação para pessoas leigas, contendo as seguintes "fases":
pc1- Criação de um arquivo.bat - Com o bloco de notas é nessesario completar o codigo para executar um arquivo que abra o CMD.
pc2- Com o CMD aberto, é necessario usar um codigo que abra o visualcode.
pc3- Na tela do Visualcode, há um codigo com um erro de digitação, corrija para executar o site.
pc4- Leia atentamente o conteudo do site para achar a chave para resetar o antivirus e clique no botão.
pc5- Insira o codigo e finalize a dinamica.
pc6- Timer para dificultar o jogo, tambem contendo um chatbot para ajudar a passar das fases.
